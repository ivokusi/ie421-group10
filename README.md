# group_10_project

## Installation Instructions

```bash
git clone https://[NETID]@gitlab.engr.illinois.edu/ie421_high_frequency_trading_fall_2025/ie421_hft_fall_2025_group_10/group_10_project.git
```

Then open Fusion360 and navigate to

UTILITIES > ADD-INS > Scripts and Add-Ins... > PLUS > Script or add-in from Device > Select ... folder

## Overview

**Part 1**: Create a Fusion360 script using Python to generate G-Code for a CNC to mill a resonant cavity for an atomic clock.
